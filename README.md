# 使用axum开发telegram机器人

[本专题](https://axum.rs/subject/tg-bot)将带你使用axum实现一个 Webhook 形式的 Telegram 机器人。

## 知识点

- 从 rust 中发送网络请求，并解析请求结果

- Telegram 机器人的几种消息类型

## 目标

我们最终将实现几个指令：

- `/website`：访问axum中文网官方

- `/logo`：查看axum中文网LOGO

- `/help`：显示帮助信息

本专题代码在[axumrs/tg-bot](https://github.com/axumrs/tg-bot)。
